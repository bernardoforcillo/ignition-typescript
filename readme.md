# Yet another monorepo starter project!

## Local development

Add the following `.env` file to this folder.

```bash

## Postgress Enviroment

POSTGRES_USER=root
POSTGRES_PASSWORD=toor
PGDATA=/var/lib/postgresql/data/pgdata

# TURBO REPO 

TURBO_TOKEN=*****
TURBO_TEAM=*****

```
